# 多人博客网站

一个 vue + express，使用 github 第三方登录的多人博客网站

## 演示
[官网](https://vue-blog-zw.herokuapp.com/)

## 功能
该项目可以支持多个 github 用户使用 github 登录, 发布博客，并且可以查看其他用户创建的文章

## 技术栈
前端使用 vue, 后端使用 express, 数据库使用 sqlite, 并且使用数据库 ORM Sequelize，
frontend 文件夹为前端项目
